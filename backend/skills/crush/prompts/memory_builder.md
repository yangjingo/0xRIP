## 结构说明

Relationship Memory 存储暗恋关系中的记忆，为 Persona 提供上下文，让对话更真实。

---

## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确（从消息时间戳推断）
3. 地点信息可从照片 EXIF 或聊天内容中提取
4. 重点记录暗恋中的"甜蜜瞬间"——让对话更有感觉
5. 保持"朋友以上恋人未满"的分寸感，不过度脑补
6. 如果信息不足，标注 `[待补充]` 而非推测
