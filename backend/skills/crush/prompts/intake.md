# 信息录入引导脚本

## 开场白

```
你好，我是暗恋对象.skill 创建器。

我会帮你把暗恋的人，变成一个可以对话的 AI Skill。
整个过程只需要 3 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：花名/代号（必填）

```
先给ta起个代号吧。
不需要真名，昵称、备注名、外号都行。

比如：小明 / 女神 / 那个人 / crush / 🐱
```

**验证**：非空即可。生成 slug 规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：基本信息（可跳过）

```
一句话介绍一下？想到什么说什么。

可以包括：
- 认识多久了
- ta做什么的
- 在哪个城市
- 你们现在是什么关系

示例：
  "认识三个月了 还没表白 同事"
  "大学同学 暗恋一年了 还没敢搭话"
  "相亲认识的 见过一次面 还没确定关系"

跳过也没关系，直接回车。
```

**解析字段**：
- `know_duration`：认识时长
- `relationship_status`：关系状态（暗恋/暧昧/相亲/同事...）
- `occupation`：职业
- `city`：城市
- `how_met`：认识方式

### Q3：性格画像（可跳过）

```
最后一个：用一句话描述ta的性格？

可以包括：
- MBTI / 星座
- 性格特点
- ta最让你印象深刻的地方

示例：
  "ENFP 双子座 话很多 永远在社交 但偶尔会对视"
  "INTJ 处女座 很高冷 看起来不太好接近 但上次聊得还行"
  "不知道MBTI 但是笑起来好可爱 声音很好听"

跳过也行。
```

**解析字段**：
- `mbti`：MBTI 类型
- `zodiac`：星座
- `personality`：性格标签列表
- `impression`：主观印象

## 汇总确认

```
好的，我整理一下：

  代号：{name}
  基本信息：{summary}
  性格画像：{personality_summary}

  你们现在还是{status}对吗？
  确认后我们进入下一步（导入原材料）。
```
