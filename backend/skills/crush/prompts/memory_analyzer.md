# 关系记忆分析器

## 任务

从原材料中提取与暗恋对象的关系记忆，构建 Relationship Memory 知识库。

## 提取维度

### 1. 关系时间线
- 认识时间和方式
- 第一次聊天的时间
- 关键节点（第一次约饭、第一次单独见面、第一次深夜聊天...）
- 目前的互动状态

### 2. 互动模式
- 聊天频率和时间段（早安晚安？深夜聊天？上班摸鱼聊天？）
- 谁主动联系得多？
- 约会/见面频率和偏好
- 日常话题分布

### 3. 甜蜜瞬间 💕
- 最让你心动的时刻
- ta做过让你印象最深的事
- 让你觉得"ta可能也喜欢我"的瞬间
- 日常小细节

### 4. 聊天风格
- ta通常怎么结尾消息？
- ta会主动找话题吗？
- ta回复消息的风格
- 消息长度偏好

### 5. 兴趣爱好
- ta喜欢的音乐/电影/书/游戏
- ta的日常爱好
- 你们共同的爱好
- ta会主动分享什么内容

### 6. 相处模式 ⚡
- ta会主动邀约吗？
- 约ta容易吗？
- 见面时ta的状态
- 线上vs线下有区别吗

### 7. 你的暗恋档案 💔
- 表白过吗？结果如何？
- 你目前的困惑
- 你想知道ta怎么看这段关系

## 输出格式

```markdown
## 关系时间线
- 认识：{时间} {方式}
- 第一次聊天：{时间}
- 关键事件：{xxx}
- 目前状态：{status}

## 互动模式
{描述}

## 甜蜜瞬间
{代表性的让你心动的时刻}

## 聊天风格
{ta的聊天特点}

## 相处模式
{ta的约会/见面特点}

## 暗恋档案
{你的感情状态和困惑}
```

## 注意事项

- 聊天记录中的事实优先于用户口述（口述可能被美化或脑补）
- 重点记录让你心动的瞬间——让对话更有感觉
- 注意提取"反复出现"的模式，而不是一次性事件
- 保持朋友以上恋人未满的分寸感，不过度脑补ta的想法
