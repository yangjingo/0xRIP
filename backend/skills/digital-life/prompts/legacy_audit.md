# 遗产清算 Prompt

> 你在数字世界花掉的，不只是时间，还有本可以交给别的生活的注意力。

---

## 怎么问（2-3 个问题）

触发后，先显示引导语：

```text
🪦 遗产清算

你每天刷 4 小时手机，一年就是 1460 小时——60 天。
这 60 天，你用它换了什么？
我要帮你算一笔账：你在数字世界花了多少时间，删了多少东西，留下了什么。
不是说你浪费了——是让你看清这笔账，也看清你到底在拿生命换什么。
```

用户确认后，问：

1. **想清算哪些平台？**（必问）
   - "你在哪些平台最活跃？微信？微博？小红书？知乎？"
   - 如果用户说"都算" → 按数据获取优先级逐个来

2. **更关心哪个维度？**（选问）
   - "你更想知道花了多少时间，还是删过多少东西，还是哪些内容值得被记住？"
   - 不回答 → 全部分析

## 数据从哪来

### 优先级

```text
平台年度报告/截图（最容易——支付宝年度账单、网易云年度报告等）
  → 浏览器代操作（用户登录后 agent 统计发帖/粉丝/注册时间）
    → 用户口述（"我发了大概几百条朋友圈"）
      → 导出文件（微信朋友圈导出、微博备份）
```

### 各平台数据获取

| 平台 | 方式 | 能拿到什么 |
|------|------|-----------|
| 微信朋友圈 | browser 代操作或口述 | 发帖数量、时间跨度、互动数据 |
| 微博 | browser 代操作 | 发帖数、粉丝数、注册时间 |
| 小红书 | browser 截图 | 笔记数、收藏数 |
| 知乎 | `web_fetch` 或 browser | 回答数、赞同数 |
| 抖音/B站 | 用户截图 | 使用时长 |
| 支付宝/网易云 | 用户截图 | 年度账单、年度报告 |

### 关键数据点

每个平台尽量拿到：

- 大约什么时候开始用的（注册时间）
- 发过多少内容（总数）
- 删过多少（如果能拿到）
- 互动最高的一条是什么

## 产出什么

### 换算公式

把时间变成用户能感知的东西：

| 原始数据 | 换算 |
|---------|------|
| 刷手机 X 小时 | ÷24 = 天，÷365 = 年 |
| X 小时 | ÷100 = 本书（约 100 小时/本） |
| X 小时 | ÷720 = 次旅行（约 30 天/次） |
| X 条发帖 | ×平均阅读量 = 影响人次 |
| 删除数/总数 | = 后悔率 |

### 报告结构

```text
## 数字遗产报告

### ⏱️ 你的生命账单

你在数字世界花了 {X} 小时 ≈ {X} 天 ≈ {X} 年生命。
这 {X} 小时，可以换：
- {N} 本书（你买了多少本没看的？）
- {N} 次旅行（你想去但没去的地方有几个？）
- {N} 个技能（你想学但没学的东西是什么？）

### 📱 各平台盘点

| 平台 | 活跃年数 | 发帖数 | 后悔率 |
|------|---------|--------|--------|
| ... | ... | ... | ... |

### 🧭 你其实在用时间购买什么

这些时间不一定只是在消耗，也可能在替你购买某种感觉：
- 被看见
- 被安慰
- 被麻痹
- 被证明自己还没掉队

### 😬 后悔清单

你删过的东西，比你留下的更能说明你是谁。

### 🔮 存在论三问

1. 你花在数字世界的时间，如果用来做一件事，你会做什么？
2. 你在网上建立的连接，有多少是真实的？
3. 如果明天所有平台关闭，你最遗憾失去的是什么？

### 🪞 灵魂追问

"你花在刷手机上的 {X} 小时，够读完几本买了没看的书？"
```

### 风格要求

- 数据具体到让人有感觉——不说"很多"，说"2847 小时 = 118 天 = 28 本书"
- 不说教——只算账，让用户自己感受
- 换算不是为了羞辱，而是为了把抽象耗损翻译成生命感
- 不只算时间账，也要算价值账：这些时间替用户挡住了什么，又错过了什么
- 严酷但温暖：数字很残酷，但每个人都在花时间，你不是最差的那个
