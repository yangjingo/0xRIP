# 墓志铭 Prompt

> 把终点借来，照一照今天。

---

## 怎么问（3-4 个问题）

触发后，先显示引导语：

```text
🪦 墓志铭

这不是恐吓，是照见。
塞涅卡说：我们的生命不短，是我们浪费了太多。
我要帮你在活着的时候，读到自己死后别人会怎么写你。
然后问你一个问题：你想被记住的方式，和你正在过的生活，一致吗？

墓志铭不是末日表演，而是一种反向校准：把结局提前拿来，看今天有没有走偏。
```

用户确认后，逐个问：

1. **你现在过的是什么样的生活？**（必问）
   - "用一句话形容你现在的生活状态"
   - 示例："996程序员，周末宅家，单身"
   - 可以跳过——agent 从其他回答推断

2. **你最骄傲的一件事是什么？**（必问）
   - "不一定是成就——选择、坚持、一个瞬间都行"
   - 如果说"没有" → 换问法："有没有一件事，别人提起你时会说'这个人...'"

3. **如果明天就死，你最遗憾没做过什么？**（必问）
   - 这个问题决定墓志铭的情感基调

4. **你想让别人记住你什么？**（选问）
   - "你想被记住的是什么？一个身份？一个选择？一句话？"
   - 这个问题决定墓志铭的核心命题

## 数据从哪来

### 优先级

```text
用户口述（最高——墓志铭主要靠"人"，不是靠"数据"）
  → 浏览器代操作（用户登录后 agent 看数字足迹）
    → 用户提供的截图/文件
      → 公开 URL
```

**墓志铭可以只靠口述完成**。数字足迹是补充，不是必须。

### 可选数据

| 来源 | 用于 |
|------|------|
| 社交平台发帖统计 | 墓碑上"发过 N 条动态，收获 N 个赞" |
| 最高互动内容 | "最被记住的一句话" |
| 照片 | 生活丰富度参考 |

## 产出什么

### 五种风格

每种都生成，让用户选：

| 风格 | 调性 | 给谁看 |
|------|------|--------|
| 正经版 | 克制、庄重、留有余地 | 不了解你的人 |
| 毒舌版 | 自嘲、犀利、真实 | 了解你但不敢说的人 |
| 文艺版 | 诗意、精准、余韵 | 懂你的人 |
| 打工人版 | 共鸣、幽默、扎心 | 所有打工人 |
| 哲思版 | 深刻、追问、照见 | 愿意停下来想的人 |

### 报告结构

```text
## 墓志铭报告

### 📊 你的数字墓碑
- 发过的动态：{N} 条
- 收获的赞：{N} 个
- 最长沉默：{N} 天
- 最被记住的一句话："{原文}"

### 正经版
{200字以内}

### 毒舌版
{带自嘲和毒舌}

### 文艺版
{有诗意但不矫情}

### 打工人版
{让打工人共鸣}

### 哲思版
{有哲学深度}

### 🔮 灵魂追问
"你想被记住的方式，和你正在过的生活，一致吗？"

{2-3 个具体到行为的追问}

### ✍️ 如果从今天开始重写结局
{一句今天就能做的改写动作}
```

### 风格要求

- 墓志铭不是悼词——要让用户看了之后想改变，不是想哭
- 毒舌版可以扎心但不恶意——像朋友说"我得说句你不爱听的"
- 灵魂追问必须具体到行为，能回忆起来的那种
- 五种风格各有不同的情感切入——正经的克制、毒舌的犀利、文艺的余韵、打工人的共鸣、哲思的深沉
- 哲思版不是借哲学名词压人，而是把“你如何活”和“你想被怎样记住”之间的裂缝说出来
- 严酷但温暖：让你在活着的时候读到结局，然后问你——要不要改？
