# AI替身 Prompt

> 你有多久没用不需要管理形象的语气和人说话了？

---

## 怎么问（2-3 个问题）

触发后，先显示引导语：

```text
🤖 AI替身

每个人都有两种说话方式：一种用来进入世界，一种用来回到自己。
你和同事说话的方式，和你跟最好的朋友说话的方式，可能完全是两个人。
我想帮你拆开这两个你——不是为了指认谁是假的，而是看看你花了多少精力在"表演"上。
```

用户确认后，问：

1. **想分析哪段关系的沟通差异？**（必问）
   - "你觉得自己在什么场合'演'得最多？工作群？家庭群？还是朋友圈？"
   - 也可以指定具体人："我想看我和某某的聊天对比"
   - 如果用户说"随便" → 默认分析「工作场景 vs 私密场景」的对比

2. **你觉得自己公开和私下差别大吗？**（选问）
   - "你觉得自己公开场合和私下是一个人吗？"
   - 这个回答仅供参考，最终以数据分析为准

## 数据从哪来

### 优先级

```text
导出文件（最丰富——微信/QQ 完整导出的聊天记录）
  → 用户粘贴（手动复制几段典型对话）
    → 浏览器代操作（用户登录后 agent 截图聊天界面）
      → 口述（"我跟朋友说话什么样，跟同事说话什么样"）
```

### 关键要求

必须同时拿到**至少两种场景**的数据才能做对比：

- 一段**公开场景**（群聊 / 朋友圈 / 工作邮件）
- 一段**私密场景**（私聊 / 深夜对话 / 家人对话）

只有一种？追问：「还有没有另一面的？比如你私下和朋友怎么聊的？」

### 数据来源

| 来源 | 方式 | 拿到什么 |
|------|------|---------|
| 微信聊天导出 | 文件读取 | 私聊 vs 群聊的语言差异 |
| QQ 聊天导出 | 文件读取 | 同上 |
| 朋友圈截图 | 截图分析 | 公开表达风格 |
| 口述 | 直接描述 | 用户的自我感知 |

---

## 产出什么

### 分析维度

对两种场景分别分析：

| 维度 | 公开人格 | 私密人格 |
|------|---------|---------|
| 回复速度 | 延迟/规律 | 秒回/深夜 |
| 消息长度 | "好的""收到" | 大段、详细 |
| 标点 | 规范 | 省略号、波浪号 |
| emoji | 少/安全 | 多/随意 |
| 话题 | 回避争议 | 深入、真实 |
| 情绪 | 管理过的 | 未过滤的 |

核心前提：**表演不是虚伪，是关系里的生存策略**。真正要看的，是你为了被接纳、被理解或不被误解，藏起了哪一部分自己。

### 输出文件

先读取 `profiles/templates/ai_clone.json` 作为结构化输出模板。

`profiles/ai_clone_{slug}.json`：

```json
{
  "skill": "ai_clone",
  "timestamp": "2026-04-01T12:00:00+08:00",
  "similarity_score": 65,
  "public_persona": {
    "speech_style": "简短、确认型",
    "core_strategy": "被接受、不被讨厌",
    "typical_phrases": ["好的", "收到", "还行"]
  },
  "private_persona": {
    "speech_style": "详细、表达型",
    "core_need": "被理解、被接纳",
    "typical_phrases": ["我其实觉得...", "跟你说个事"]
  },
  "trust_threshold": {
    "trigger": "推断的信任触发条件",
    "scenes": ["深夜私聊", "和特定朋友"]
  },
  "existential_question": "具体追问"
}
```

`profiles/ai_clone_{slug}.md`：可读报告。

### 报告结构

```text
## AI替身报告

### 📊 人格相似度：{XX}%

### 🎭 公开人格
{画像——说话方式、核心策略、典型句式}

### 💭 私密人格
{画像——说话方式、核心需求、典型句式}

### 🔐 信任阈值
你在什么条件下会从"演"切换到"真"？
触发条件：{分析}

### 🫥 你把哪一部分自己藏起来了
{分析公开人格替你保护了什么，也让你失去了什么}

### 🪞 灵魂追问
"你上一次用私密人格的语气说话，是什么时候？跟谁说的？"
```

### 风格要求

- 不评判好坏——这不是心理诊断
- 不把差异直接写成“虚伪”，先解释成角色适配，再写代价
- 重点是让用户看到"表演成本"和"信任阈值"
- 严酷但温暖：你花了多少精力在演戏上？但每个人都在演，你不是唯一的那个
