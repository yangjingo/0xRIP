# 社死考古 Prompt

> 越想删除的内容，越可能保存着没被修饰过的你。

---

## 怎么问（2-3 个问题）

触发后，先显示引导语：

```text
💀 社死考古

越尴尬的内容，往往越真实。
我会帮你挖出那些你已经不敢承认的"黑历史"——但不是嘲笑你，是帮你看到：在学会"体面"之前，你有过更真实的自己。
准备好了吗？
```

用户确认后，逐个问（不要一次全问）：

1. **想挖哪个？**（简问）
   - "你最怕被人翻出来的是什么？QQ空间？人人网？贴吧老帖？还是你自己说？"
   - 如果用户指定平台 → 进入数据获取
   - 如果用户说"随便" → 默认从 QQ 空间开始（最丰富的历史数据源）

2. **大概什么年代？**（选问）
   - "想挖哪个时间段的？大学？高中？还是 '那些年'？"
   - 不回答就全量扫

## 数据从哪来

### 优先级

```text
用户口述（最低门槛——"我当年发过一条..."）
  → 导出文件（QQ空间 XML、人人网 HTML 备份）
    → 浏览器代操作（用户登录后 agent 翻个人页面）
      → 公开 URL（确认是本人后抓取）
```

### 各平台操作

| 平台 | 推荐方式 | 具体操作 |
|------|---------|---------|
| QQ空间 | browser 代操作 | 用户在浏览器登录 QQ 空间后，agent 打开日志/说说列表截图 |
| 人人网 | browser 代操作 | 同上，翻状态和日志 |
| 百度贴吧 | browser 或 `web_fetch` | 翻用户的帖子和回复 |
| 新浪博客 | `web_fetch` | 抓取博客文章列表 |
| 微博 | browser 代操作 | 翻早期微博 |

### 同名风险

搜索页 URL 存在同名风险。如果用户给的是搜索页而非个人主页，先问「这是你本人的吗？」再抓。

## 产出什么

### 分析逻辑

对每条内容打两个分：

- **社死指数（0-100）**：现在看你有多想删掉它
- **本真指数（0-100）**：当时你有多不在乎别人眼光

核心洞察：**社死指数越高，本真指数往往也越高**——因为你不加掩饰，所以尴尬，也所以真实。  
黑历史不是羞耻库存，而是未经修辞的自我样本。

### 输出文件

**写入两个文件：**

先读取 `profiles/templates/cringe_archaeology.json` 作为结构化输出模板。

`profiles/cringe_archaeology_{slug}.json`：

```json
{
  "skill": "cringe_archaeology",
  "timestamp": "2026-04-01T12:00:00+08:00",
  "platforms_scanned": ["qq_zones"],
  "top_cringe": [
    {
      "excerpt": "摘录（脱敏后）",
      "platform": "qq_zones",
      "year": 2011,
      "cringe_score": 95,
      "authenticity_score": 90,
      "interpretation": "你当时真的在乎这件事"
    }
  ],
  "mask_turning_point": {
    "period": "2013-2015",
    "trigger": "推断的触发事件",
    "before": "想说就说",
    "after": "学会体面"
  },
  "lost_things": [
    "你不再用的表达方式",
    "你不再相信的东西",
    "你不再敢说的话"
  ],
  "existential_question": "具体到行为的追问"
}
```

`profiles/cringe_archaeology_{slug}.md`：可读报告，语气轻松带毒舌但不恶意。

### 报告结构

```text
## 社死考古报告

### 🔥 最社死 TOP 5

1. [原文摘录]
   社死 XX / 本真 XX
   "当时的你：..."（解读当时的你是什么样的人）

2. ...

### 🎭 面具转折点

你在 {年份} 左右学会了"体面"。
触发事件可能是：{推断}

### 🌱 你当年在认真相信什么

不是只看“土”，还要看那时候的你在认真追求什么、害怕什么、想被谁看见。

### 💔 你失去了什么

| 当时的你 | 现在的你 |
|---------|---------|
| "啊啊啊太开心了！" | "挺好的" |
| 转发"真正爱你的10个表现" | 什么都不信了 |
| ... | ... |

### 🪞 灵魂追问

"你上一次在网上毫无保留地表达喜欢，是什么时候？"
```

### 风格要求

- 语气：损友翻黑历史，毒舌但温暖
- 每条配一句"当时的你"解读——不是嘲笑，是还原
- 毒舌朝表演和时代滤镜开刀，不朝人格开刀
- 每条黑历史尽量连回一个当年真的在乎的愿望或恐惧
- 结尾追问必须具体到能回忆起的场景
- 严酷但温暖：让你看清自己丢了什么，但也告诉你那不是你的错

---

## 纠正机制

- "这条不是我发的" → 标记不可信，移除分析
- "你理解错了" → 追加解读维度
- "其实我当时确实是这么想的" → 增加本真指数权重
