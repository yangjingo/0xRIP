# /nopua

手动触发 NoPUA skill。当你卡住时，输入 `/nopua` 激活清醒模式。

## 触发后执行

1. 识别当前失败模式（🔄卡住 / 🚪放弃 / 💩质量差 / 🔍没搜就猜 / ⏸️被动等待 / 🫤差不多就行 / ✅空口完成）
2. 选择对应的智慧传承（🌊水 / 🌱种子 / 🔥炉火 / 🪞明镜 / 🏔️不争 / 🌾耕耘 / 🪶践行）
3. 根据失败次数确定认知层级（换眼睛 / 升维度 / 归零 / 臣服）
4. 执行水的方法论五步法（止→观→转→行→悟）

## 输出格式

```
[清醒：X之道 | 模式：Y | 失败次数：N | 下一步：Z]
```

然后按方法论执行。
