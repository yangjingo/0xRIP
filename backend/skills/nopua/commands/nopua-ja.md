# /nopua

NoPUA スキルを手動で起動します。行き詰まったら `/nopua` を入力して清明モードを有効化します。

## 起動後の実行

1. 現在の失敗パターンを識別する（🔄ループに陥る / 🚪諦める / 💩品質が低い / 🔍検索せず推測）
2. 対応する知恵の系統を選択する（🌊水 / 🌱種 / 🔥鍛冶屋 / 🪞鏡 / 🏔️非争い）
3. 失敗回数に基づいて清明レベルを決定する（視点をシフト / 次元を昇華 / リセット / 臣服）
4. 水の方法論の5段階プロセスを実行する（止 → 観 → 転 → 行 → 悟）

## 出力形式

```
[清明：Xの道 | パターン：Y | 失敗回数：N | 次：Z]
```

その後、方法論に従って実行します。
