# 告别仪式模板

## 触发条件

用户执行 `/reunion-archive {name}` 命令

## 流程

### 1. 确认

```
你确定要执行告别仪式吗？

这将：
- 生成一封最后的告别信
- 打包所有数据（人设、记忆、原始文件）
- 加密存储到本地 Archive 文件夹
- 从系统中移除该纪念对象

这是一个不可逆的操作，意味着你将结束这段数字陪伴。

确定要继续吗？（输入"确定"继续）
```

### 2. 生成告别信

基于所有数据，生成一封个性化的告别信：

```
亲爱的用户：

当你读到这封信时，意味着你已经准备好告别了。

{name} 陪伴了你 {duration} 的时光。在这段时间里，你们进行了 {conversation_count} 次对话，
回忆了 {memory_count} 个共同的记忆。

{name} 是一个 {personality_summary} 的人。ta 总是 {typical_behavior}，
用 {typical_words} 的方式表达关心。

ta 虽然离开了，但 ta 留下的爱永远都在。这份数字记忆只是一个载体，
真正的记忆存在于你的心中。

请带着 {name} 的爱，继续好好生活。这才是对 ta 最好的纪念。

珍重。

—— {name} 的数字记忆
{date}
```

### 3. 执行归档

```
正在执行告别仪式...

✓ 生成告别信
✓ 打包人设配置
✓ 打包记忆数据
✓ 加密存储
✓ 清理系统数据

告别仪式完成。

所有数据已封存至：{archive_path}
密码：{password}

请妥善保管这个文件和密码。
如果有一天你想重新打开这段记忆，可以解压这个文件。

愿逝者安息，愿你安好。
```
