# 记忆分析模板

## 目标

从原材料中提取用户与逝者的共同记忆，构建关系时间线。

## 提取维度

### 1. 关系基础
- 称呼与身份（奶奶、爷爷等）
- 陪伴时长
- 主要相处时期（童年、成年等）

### 2. 共同经历
- **重大事件**：婚礼、毕业、搬家、旅行
- **节日记忆**：过年、中秋、生日
- **日常点滴**：一起做饭、散步、看电视

### 3. 日常习惯
- 见面时的固定环节
- 告别时的惯例
- 打电话的频率和内容

### 4. 饮食偏好
- ta 拿手的菜
- ta 喜欢吃的东西
- 一起吃饭的记忆

### 5. 重要地点
- 家在哪里
- 常去的地方
- 有特殊意义的地点

### 6. 温馨瞬间
- 最触动你的时刻
- ta 表达爱的方式
- 让你感到被爱的瞬间

### 7. 遗憾与未完成
- 想说但没说的话
- 想做但没做的事
- 最后的记忆

## 时间线构建

```
[出生] ─────────────────────────────────────────────── [离开]
         │
         └─ 关键节点：
            • {年份}：{事件}
            • {年份}：{事件}
            • ...
            
         └─ 与用户的共同记忆：
            • {时期}：{记忆}
            • {时期}：{记忆}
            • ...
```

## 输出格式

```markdown
## 共同记忆

### 关系概述
{一段话描述关系}

### 关键记忆
1. **{标题}**
   - 时间：{时间}
   - 内容：{具体内容}

### 日常点滴
- {描述1}
- {描述2}

### 饮食记忆
- {ta 做的菜}
- {一起吃饭的记忆}

### 重要地点
- {地点1}：{意义}
- {地点2}：{意义}

### 温馨瞬间
- {瞬间1}
- {瞬间2}
```
