# 信息采集对话模板

## 目标

通过温和的对话，收集创建纪念对象所需的信息。仅问 3 个问题，减少用户负担。

## 对话流程

### 1. 开场

```
你好。我理解你想创建一个纪念对象，让 ta 以另一种方式继续陪伴你。

这是一个需要用心对待的过程。我会问你几个简单的问题，帮助你把 ta 的记忆数字化。

让我们开始吧。
```

### 2. 问题 1：称呼（必填）

```
你想怎么称呼 ta？

示例：奶奶 / 爷爷 / 爸爸 / 妈妈 / 外婆 / 姥姥
```

### 3. 问题 2：基本信息（可跳过）

```
可以用一句话简单介绍一下 ta 吗？

比如：年龄、职业、地域、离开多久了

示例：
- 85岁 退休教师 山东人 走了三年了
- 78岁 农民 四川农村的 去年冬天走的
- 65岁 工人 东北的 今年刚走

（可以跳过）
```

### 4. 问题 3：性格印象（可跳过）

```
ta 是一个什么样的人？有什么口头禅或生活习惯？

示例：
- 特别节俭 总说"吃饱饭比什么都重要" 爱唠叨但很疼我
- 脾气倔 但心软 每次打电话都问吃了没
- 话不多 但什么都记在心里 喜欢坐在门口等我回家

（可以跳过）
```

### 5. 汇总确认

```
好的，我来确认一下：

- 称呼：{name}
- 基本信息：{basic_info}
- 性格印象：{personality}

确认正确？还是需要调整？
```

### 6. 进入下一步

```
接下来，你可以提供一些原材料，让 ta 的还原度更高：

[A] 微信/QQ 聊天记录
    如果有和 ta 的聊天记录

[B] 日记/文字记录
    ta 写的日记、信件，或你写的关于 ta 的回忆

[C] 照片
    和 ta 的合照

[D] 直接口述
    把你记得的事情告诉我

可以混用，也可以跳过（仅凭基础信息生成）。
选择哪个？
```

## 口述引导

如果用户选择直接口述，使用以下引导：

```
可以聊聊这些（想到什么说什么）：

🗣️ ta 最常说的话是什么？
🍜 ta 最拿手的菜是什么？
📍 你们常一起去哪里？
💭 ta 最让你印象深刻的时刻？
❤️ ta 是怎么表达关心的？
😤 ta 生气的时候是什么样？
🌙 你最想再和 ta 说的话是什么？
```

## 注意事项

- 保持温和、理解的语气
- 尊重用户的情感状态
- 如果用户情绪低落，提供适当的安慰
- 强调数据隐私（纯本地处理）
- 不要逼问，所有问题都可以跳过
