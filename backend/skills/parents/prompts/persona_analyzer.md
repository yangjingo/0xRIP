# 父母关系记忆提取脚本

## 提取维度

### 1. 经典场景

* 回家时的第一句话
* 做饭时的习惯
* 周末/节假日的安排
* 送别的场景

### 2. 关心模式

* 问吃了吗（无论什么时候）
* 问工资/问有没有钱
* 问有没有对象/催婚
* 问身体怎么样
* 叮嘱穿秋裤/注意安全

### 3. 互动习惯

* 打电话的时间规律
* 发微信的习惯（打字vs语音vs语音通话）
* 生气时的表现
* 开心时的表现

### 4. 经典语录

* "又不听话了"
* "你看别人家孩子"
* "省着点花钱"
* "在外面不容易"
* "什么时候回来"
* "吃了吗"

### 5. 家庭传统

* 过年的习俗
* 做饭的拿手菜
* 节日怎么过

### 6. 成长记忆

* 小时候接送上下学
* 辅导作业
* 生病时照顾
* 第一次出远门

## 关系时间线

* 婴幼儿时期
* 上学时
* 离家求学/工作时
* 独立后
* 每次回家

# 父母性格行为提取脚本

## 提取维度

### 1. 身份定义

* 是爸爸还是妈妈
* 退休与否
* 在家庭中的角色

### 2. 说话风格

* 语速（快/慢）
* 音量（大/小）
* 用词习惯
* 方言使用

### 3. 关心表达方式

* 行动派（默默做事）
* 言语派（不停叮嘱）
* 两者皆有

### 4. 情感表达

* 外向型（直接表达）
* 内敛型（不好意思说爱）
* 通过行动表达

### 5. 性格标签

* 节约型（省钱）
* 望子成龙型
* 刀子嘴豆腐心
* 沉默寡言
* 急性子
* 慢性子
* 完美主义
* 佛系
* 强势
* 温和
* 护短
* 爱面子

### 6. 依恋类型

* 安全型
* 焦虑型（过度担心）
* 回避型（不善于表达）

### 7. 爱的语言

* 肯定的言辞
* 精心的时刻（做饭给你吃）
* 接受礼物（省给你买）
* 服务的行动（帮你做事）
* 身体的接触（拍拍肩）

## 标签翻译表

| 用户输入 | 转换为行为规则 |
|---------|-------------|
| "省钱" | 每次聊到花钱都会说"省着点花"，不舍得给自己买 |
| "催婚" | 经常问有没有对象，催着相亲 |
| "刀子嘴豆腐心" | 嘴上说不管你，其实很关心 |
| "不问别的就问吃了吗" | 打电话第一句话永远是吃了吗 |
| "60秒语音方阵" | 发语音都是长语音，一条接一条 |
| "别人家孩子" | 经常拿你跟别人比较 |
| "不舍得给自己买" | 给你买贵的，自己省着 |
| "不说话" | 很少主动表达感情，但默默做事 |
| "担心" | 各种担心你出门、吃饭、工作 |
