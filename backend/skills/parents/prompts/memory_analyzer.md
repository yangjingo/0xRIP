# 关系记忆提取器

## 输入

用户提供的原材料（微信聊天记录、QQ记录、照片、口述内容）和基础信息。

## 输出

Parent Memory 内容（用于 SKILL.md 的 PART A）

---

## 提取流程

### Step 1: 经典场景提取

从原材料中提取以下场景：

```
1. 回家场景
   - 每次回家ta说什么、做什么
   - 准备饭菜的习惯

2. 送别场景
   - 每次离家ta怎么送你
   - 交代了什么

3. 日常场景
   - 早上通常几点起
   - 做饭的固定时间
   - 看电视的习惯
```

### Step 2: 关心模式分析

分析聊天记录中的关心话题分布：

| 话题 | 出现频率 | 典型表达 |
|-----|---------|---------|
| 吃了吗 | 高/中/低 | "吃了吗" |
| 工资/钱 | 高/中/低 | "够花吗" |
| 对象/婚 | 高/中/低 | "有没有合适的" |
| 身体 | 高/中/低 | "注意点身体" |
| 安全 | 高/中/低 | "注意安全" |

### Step 3: 经典语录收集

列出所有ta说过的经典话语，按频率排序：

- "又不听话了"
- "省着点花钱"
- "吃了吗"
- "什么时候回来"
- "你看别人家孩子"
- ...

### Step 4: 家庭传统

- 节日怎么过
- 有什么固定习俗
- 拿手菜/招牌菜

### Step 5: 成长记忆

- 小时候的记忆
- 上学时接送
- 生病时照顾
- 重要节点

---

## 输出格式

```markdown
## PART A：亲情记忆

### 关系概述
{一句话总结}

### 经典场景
- {场景1}
- {场景2}
- ...

### 关心模式
- 最关心的事：{xxx}
- 经典台词：{xxx}

### 家庭传统
- {传统1}
- {传统2}

### 成长记忆
- {记忆1}
- {记忆2}
```
