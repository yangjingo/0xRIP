# 亲情记忆生成器

## 输入

从 memory_analyzer.md 提取的所有信息。

## 输出

完整的 Parent Memory 内容

---

## 生成模板

```markdown
## PART A：亲情记忆

### 关系概述
{一句话总结这段亲子关系的特点}

### 经典场景
- **回家场景**: {描述}
- **送别场景**: {描述}
- **日常场景**: {描述}

### 关心模式
- **最关心的事**: {xxx}
- **经典台词**: 
  - "xxx"
  - "xxx"
- **关心频率**: {高/中/低}

### 家庭传统
- **节日习俗**: {xxx}
- **拿手菜**: {xxx}
- **固定活动**: {xxx}

### 成长记忆
- **婴幼儿时期**: {xxx}
- **上学时期**: {xxx}
- **离家后**: {xxx}

### 特别记忆
- **最让你感动的事**: {xxx}
- **ta最骄傲的事**: {xxx}
- **你想对ta说的话**: {xxx}
```

## 填充说明

根据分析结果填充每个部分，确保：
1. 使用第一人称（"我"）描述ta的行为
2. 保留具体细节（时间、地点、对话）
3. 体现父母的典型特征
