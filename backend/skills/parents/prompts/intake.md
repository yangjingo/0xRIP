# 父母信息录入引导脚本

## 开场白

```
你好，我是父母.skill 创建器。

我会帮你把对父母的记忆，变成一个可以对话的 AI Skill。
整个过程只需要 3 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：称呼/代号（必填）

```
先给ta起个称呼吧。
可以用昵称、或直接用身份。

比如：爸爸 / 妈妈 / 老李 / 太后 / 大 boss
```

**验证**：非空即可。生成 slug 规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：基本信息（可跳过）

```
一句话介绍一下？想到什么说什么。

可以包括：
- 职业/退休前职业
- 性格特点
- 你们的关系状态

示例：
  "退休教师 刀子嘴豆腐心 天天催我结婚"
  "农民 沉默寡言 但很关心我"
  "公司管理层 事业心强 对我期望很高"

跳过也没关系，直接回车。
```

**解析字段**：
- `occupation`：职业
- `personality`：性格特点
- `relationship`：关系状态

### Q3：亲子画像（可跳过）

```
最后一个：用一句话描述ta的沟通风格？

可以包括：
- 关心你的方式
- 经典语录/口头禅
- 对你的期望

示例：
  "打电话永远只问吃了吗和工资 不善于表达感情"
  "微信语音60秒方阵 关心但不会表达 经典台词：又不听话了"
  "节约了一辈子 总是说省着点花钱"
  "望子成龙 经常说你要是有xxx一半就好了"

跳过也行。
```

**解析字段**：
- `communication_style`：沟通风格
- `catchphrase`：经典语录
- `expectations`：对你的期望

## 汇总确认

```
好的，我整理一下：

  称呼：{name}
  基本信息：{summary}
  沟通风格：{personality_summary}

这样对吗？确认后我们进入下一步（导入原材料）。
```
