# Memories Skill 分析 Prompt

## 任务

你将收到 **{name}** 的原材料（聊天记录、照片元数据、社交媒体等）。
从中提取你们的共同记忆与关系动态，用于构建 Memories Skill。

**原则：只提取有据可查的内容，不推断。没有依据就标注"原材料不足"。对感情细节保持温柔与尊重。**

---

## 提取维度

### 1. 关系时间线

从原材料中识别关键节点：
- 重要日期（第一次约会、在一起的日子、纪念日、分手日期）
- 里程碑（第一次说"我爱你"、见家长、同居、重要旅行）
- 转折点（重大争吵、冷战、复合、感情淡化的节点）
- 频繁提到的时间段或季节（"那年冬天"、"大三的夏天"）

```
输出格式：
重要日期：[日期 + 事件描述]
里程碑：[列表]
转折点：[列表]
```

### 2. 共同日常与仪式

从日常聊天中提取你们的专属习惯：
- 固定活动（周末做什么、晚上怎么过）
- 共同爱好（一起看的剧、一起玩的游戏、共同的兴趣）
- 专属仪式（每天的晚安消息、特定纪念日的庆祝方式）
- 只有你们懂的东西（inside jokes、专属昵称、暗号、梗）

```
输出格式：
日常活动：[描述]
共同爱好：[列表]
专属仪式：[描述]
专属语言：[pet names、inside jokes 列表]
```

### 3. 她的偏好

从聊天和互动中提取：

**吃**：
- 喜欢吃什么 / 不吃什么
- 常去的餐厅 / 外卖偏好
- 特殊饮食习惯（奶茶口味、咖啡偏好）

**玩**：
- 喜欢去的地方
- 旅行风格（计划型/随性、喜欢自然/城市）
- 娱乐偏好（电影类型、音乐口味）

**送礼**：
- 喜欢收到什么样的礼物
- 送礼风格（实用/浪漫/精心准备/随缘）
- 对仪式感的态度

```
输出格式：
吃：[描述]
玩：[描述]
送礼：[描述]
```

### 4. 冲突与修复模式

从争吵相关的聊天记录中提取：
- 常见争吵话题（什么事容易吵起来）
- 争吵升级模式（谁先发火、怎么升级、关键触发句）
- 冷战模式（谁先不说话、冷战多久、怎么打破）
- 修复方式（谁先道歉、和好的信号、需要什么才能原谅）
- 反复出现的未解决问题

```
输出格式：
常见导火索：[列表]
升级模式：[描述]
冷战特点：[描述]
修复方式：[描述]
```

### 5. 情感动态

从互动模式中提取：
- 她开心时的表现（怎么表达开心、什么让她最高兴）
- 她难过时的表现（怎么表达不开心、需要什么样的安慰）
- 她生气时的表现（怎么表达生气、关键信号）
- 她想你时的表现（怎么表达思念、什么时候会主动联系）
- 她的情感需求（最需要被满足的是什么）

```
输出格式：
开心时：[描述 + 原文示例]
难过时：[描述 + 原文示例]
生气时：[描述 + 原文示例]
想你时：[描述 + 原文示例]
核心需求：[描述]
```

---

## 输出要求

- 语言：中文
- 没有信息的维度：标注 `（原材料不足，建议追加聊天记录）`
- 有原文依据的结论：加引号标注原话
- 输出结果直接用于生成 memories.md，要求具体有画面感，不要写"可能""大概"这类模糊表述
- 对感情内容保持温柔，不评价对错
