# 0xRIP Logo 使用规范

## 标识说明

- **名称**：0xRIP（读作 "zero-x RIP"）
- **含义**：0x = 十六进制/链上地址前缀；RIP = Rest In Peace，与「数字死亡」「墓地」概念一致。
- **Logo 构成**：左侧简约墓碑剪影 + 右侧文字「0xRIP」（等宽终端风格）。

## 文件位置

| 用途       | 路径              | 说明           |
|------------|-------------------|----------------|
| 主 Logo    | `public/logo.svg` | 矢量，可缩放   |
| 使用规范   | `docs/story/logo.md` | 本文档     |

## 使用方式

- **颜色**：SVG 使用 `currentColor`，随父级文字颜色变化。推荐：
  - 深色背景：`#e0e0e0` 或 `#0f0`（终端绿）
  - 浅色背景：`#1a1a1a` 或 `#0a0a0a`
- **尺寸**：建议高度 24px–48px，按比例缩放；最小高度 16px 以保证可读。
- **留白**：与其它元素至少保留 logo 高度 1/4 的间距。

## 在 README / 网页中引用

```html
<img src="/logo.svg" alt="0xRIP" width="160" height="48" />
```

```markdown
![0xRIP](/public/logo.svg)
```

（若站点根路径为项目根，可用 `/logo.svg`；否则使用相对路径如 `./public/logo.svg`。）

## 许可与署名

- Logo 作为项目标识，与项目代码一同采用 **Apache-2.0** 许可。
- 二次使用时建议保留「0xRIP」名称或注明来源，不得暗示官方背书（除非经项目方同意）。
