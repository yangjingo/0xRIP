# 0xRIP

Digital graveyard. Bury memories, summon data souls, explore a 3D memorial world.

```bash
cd backend && bun install && bun run dev   # :8000
cd frontend && bun install && bun run dev  # :5173
npm install -g mmx-cli && mmx auth login --api-key <key>
```

[Bun](https://bun.sh) + [MiniMax M2.7](https://platform.minimaxi.com) + [React](https://react.dev) + [Three.js](https://threejs.org). Apache-2.0.
